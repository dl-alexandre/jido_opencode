## [0.1.0] - 2026-03-29

### Added
- Initial implementation of Jido.Harness.Adapter for OpenCode
- HTTP client for OpenCode server API
- Event translation from OpenCode SSE to Jido.Harness.Event
- Session management support
- Structured output (JSON Schema) support
- Usage tracking
- Cancellation support
- Full test suite with adapter contract tests
- Documentation and examples
